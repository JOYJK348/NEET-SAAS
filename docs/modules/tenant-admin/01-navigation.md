# Main Navigation

## Purpose

The main navigation provides quick access to all major modules required for managing the coaching institute.

## Navigation Structure

- Dashboard
- Students
- Parents
- Tutors
- Academics
  - Batches
  - Subjects
  - Timetable
  - Live Classes
- Mock Tests
  - Test Management
  - OMR Upload
  - Paper Evaluation
  - Results
- Study Materials
- Announcements
- Fee Management
  - Fee Structures
  - Student Fee Records
  - Payments
  - Receipts
  - Reminders
- Reports & Analytics
- Settings

## Principles

- Frequently used modules should appear at the top.
- Related modules should be grouped together.
- Navigation should be simple and easy to understand.
- The structure should support future expansion without major changes.
