# Tenant Admin

## Overview

The Tenant Admin is the Coaching Institute Owner who is responsible for managing the overall operations of a coaching institute.

The Tenant Admin does not directly teach students. Instead, this role manages the institute, students, tutors, academic activities, and monitors the day-to-day functioning of the institute.

The goal of the Tenant Admin is to ensure that the coaching institute operates efficiently and provides a smooth learning experience.

## Objectives

- Manage the overall coaching institute.
- Manage students and tutors.
- Manage academic operations.
- Monitor daily institute activities.
- Configure institute settings.
- Monitor institute performance.

## Responsibilities

- Manage Institute Operations
- Manage Students
- Manage Tutors
- Manage Academic Structure
- Manage Tests & Evaluations
- Monitor Institute Performance
- Communicate with Parents & Students
- Configure Institute Settings
- Review Reports & Analytics
- Ensure Smooth Daily Operations
