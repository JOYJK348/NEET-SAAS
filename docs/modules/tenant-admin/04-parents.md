# Parent Management

## Module Overview

The Parent Management module is responsible for managing parent information, maintaining parent-student relationships, and enabling effective communication between the coaching institute and parents.

The module ensures that parents remain informed about their child's academic journey while providing the institute with a centralized system to manage parent accounts and communication.

---

# Objectives

- Maintain accurate parent information.
- Associate parents with students.
- Enable parent portal access.
- Improve institute-parent communication.
- Support parent engagement throughout the student's academic journey.

---

# Parent Lifecycle

The complete parent journey consists of the following stages.

## 1. Parent Registration

Parent information is collected during the student admission process.

Purpose

- Capture parent details.
- Verify contact information.

---

## 2. Parent Association

The parent is linked to one or more students.

Purpose

- Establish the parent-student relationship.
- Enable academic monitoring.

---

## 3. Parent Account Provisioning

The system automatically creates the parent account.

Purpose

- Generate login credentials.
- Send welcome communication.
- Enable portal access.

---

## 4. Parent Engagement

Parents actively participate in the student's academic journey.

Examples

- View Attendance
- View Test Results
- Monitor Performance
- Receive Announcements

---

## 5. Parent Support

The institute assists parents whenever required.

Examples

- Reset Password
- Update Contact Information
- Resolve Login Issues

---

# Parent Responsibilities

Parents are expected to:

- Monitor student attendance.
- Monitor academic performance.
- Stay updated with institute announcements.
- Communicate with the institute when necessary.
- Support the student's learning journey.

---

# Parent-Student Relationship

Business Rules

- Every student must have at least one parent or guardian.
- One parent may be linked to multiple students.
- Parent accounts are automatically created after admission.
- Parent information should remain synchronized with student records.
