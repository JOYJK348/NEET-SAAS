# Parent

## Module Overview

The Parent module provides parents and guardians with a centralized portal to monitor their child's academic journey within the coaching institute.

Parents can track attendance, academic performance, mock test results, announcements, and overall learning progress without directly participating in academic activities.

The objective of this module is to improve transparency, strengthen communication between the institute and parents, and encourage active involvement in the student's academic growth.

---

# Objectives

- Monitor student academic progress.
- Track attendance.
- Review examination performance.
- Stay updated with institute announcements.
- Improve parent-institute communication.
- Support the student's learning journey.

---

# Responsibilities

A Parent is responsible for

- Monitoring the child's academic progress.
- Reviewing attendance regularly.
- Tracking mock test performance.
- Staying informed about institute announcements.
- Supporting the child at home.
- Communicating with the institute whenever required.

# Permissions

A Parent can

- View Linked Children
- View Attendance
- View Academic Performance
- View Mock Test Results
- View Study Materials
- View Announcements
- Manage Own Profile

A Parent cannot

- Create Students
- Modify Student Information
- Submit Assignments
- Join Live Classes
- Evaluate Papers
- Manage Tutors
- Manage Courses
- Access Institute Reports
- Modify Academic Structure

---

# Business Rules

- Every parent must be linked to at least one student.
- A parent may be associated with multiple students.
- Parents can access only their linked children's information.
- Parents have view-only access to academic information.
- Student admission and parent linking are managed by the institute.
- Parents cannot modify academic records.
