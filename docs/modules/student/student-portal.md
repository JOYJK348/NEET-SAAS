# Student

## Module Overview

The Student module provides students with a centralized learning platform to manage their complete academic journey within the coaching institute.

Students can access their daily schedule, attend live classes, study learning resources, complete assignments, participate in mock tests, and continuously monitor their academic performance.

The objective of this module is to provide every student with a structured, engaging, and distraction-free learning experience.

---

# Objectives

- Access daily academic activities.
- Attend classroom and live sessions.
- Access learning resources.
- Complete assignments.
- Participate in mock tests.
- Track academic performance.
- Stay informed through announcements.
- Continuously improve learning outcomes.

---

# Responsibilities

A Student is responsible for

- Attending scheduled classes.
- Joining live classes.
- Accessing assigned learning resources.
- Watching recorded classes.
- Completing assignments.
- Participating in mock tests.
- Reviewing performance reports.
- Maintaining regular attendance.
- Preparing consistently for examinations.
